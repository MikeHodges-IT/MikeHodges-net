Orlando Building Energy Benchmarking - Oracle APEX Edition

Technologies Used

Oracle APEX, PL/SQL, JavaScript, Leaflet, HTML, CSS, APEX AJAX Callbacks, Dynamic Actions

I developed a custom plugin for the Leaflet map that responds to Energy Star ratings by dynamically adjusting the size and color of the circles based on the value of the data. The plugin utilizes the page designer's SQL source, native refresh, and other Oracle APEX features.

Description of Project

A redesigned and enhanced version of the Green Works Orlando Building Energy Benchmarking program, built as part of my Oracle APEX portfolio. This project tracks and analyzes the energy and water usage of Orlando's largest buildings.

  • Created a custom plugin that dynamically changes the size and color of map points on a Leaflet map, featuring an AJAX callback with native refresh and data updates from the SQL source in the page designer.
  • Improved decision-making capabilities by integrating faceted searches, dynamic actions, and interactive reports, allowing users to filter and explore data effortlessly.
  • Integrated Leaflet maps and APEX components such as graphs and grids to deliver comprehensive visual data insights.
  • Focused on creating a user-friendly interface with custom CSS and JavaScript, ensuring a smooth, interactive experience tailored for energy data analysis.
  • Developed a responsive design for optimal usability across devices, ensuring accessibility for all users.
