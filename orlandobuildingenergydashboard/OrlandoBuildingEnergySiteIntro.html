About the Project

This project highlights the integration of multiple technologies and custom development within Oracle APEX, including:

  • Custom Plugin: A key feature is the Leaflet-powered map that dynamically changes the size and color of map points based on data. The plugin uses AJAX callbacks to update map points and interact with the database, leveraging the native region refresh functionality and pulling the query directly from the Page Designer source. This ensures that the map stays updated with the latest data without requiring page reloads.
  • Technologies Used: This project utilizes a range of technologies:
    • PL/SQL and SQL for managing and processing data.
    • HTML, CSS, and JavaScript for the front-end interface and interactivity.
    • APEX AJAX callbacks, processes, and dynamic actions to create an engaging and highly interactive experience.
    • Faceted Search and interactive reports for efficient data filtering and exploration.
  • Focus on Customization: The site features custom CSS styling and JavaScript enhancements, giving it a polished, user-friendly feel. The goal is to provide a seamless and intuitive interface for exploring energy data.
