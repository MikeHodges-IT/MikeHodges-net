About the Project

This project highlights the integration of multiple technologies and custom development within Oracle APEX, including:

  • Custom Plugin: A key feature is the Leaflet-powered map that dynamically changes the size and color of map points based on data. The plugin uses AJAX callbacks to update map points and interact with the database, leveraging the native region refresh functionality and pulling the query directly from the Page Designer source. This ensures the map stays updated with the latest data without requiring page reloads.
  • Technologies Used:
    • PL/SQL and SQL are used to manage and process data.
    • HTML, CSS, and JavaScript for the front-end interface and interactivity.
    • APEX AJAX callbacks, processes, and dynamic actions create an engaging and highly interactive experience.
    • Faceted Search and interactive reports are implemented for efficient data filtering and exploration, ensuring a productive user experience.
    • Collections are used to keep track of the comparison sites, making it easier to manage user selections and data comparisons.
  • Emphasizing Customization: The site is designed with custom CSS styling and JavaScript enhancements, giving it a polished, user-friendly feel. The goal is to provide a seamless and intuitive interface for exploring energy data, ensuring a comfortable user experience.
